# DiceBot
주사위 등의 TRPG 보조 기능이 포함된 봇입니다.

## 외부 라이브러리
discord.py==1.2.3

## 개발자
twitter `@_zer0ken_`  
discord `zer0ken#6146`

## 기능
* 주사위  
  ```
  주사위를 굴립니다. 주사위 표현식은 10D6, 1D, D100, D% 등의 TRPG식 표기법을 사용하시면 됩니다.
  ```
* 개인방
  ```
  특정 멤버만 들어올 수 있는 텍스트 채널과 음성 채널을 만듭니다.
  ```
* 화면공유
  ```
  음성 채널에서 화면공유를 활성화하는 링크를 확인합니다.
  ```
* 명령어
  ```
  이 명령어 목록을 확인합니다.
  ```
* 도움말
  ```
  명령어의 도움말을 확인합니다.
  ```
